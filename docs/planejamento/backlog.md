# Backlog do projeto

Este documento lista todas as funcionalidades planejadas para o Go Calendar, desde as mais básicas até as que podem entrar em versões futuras.

---

## Funcionalidades que estarão na versão 1.0 (entrega final — 3º bimestre)

Estas são as funcionalidades obrigatórias que o sistema precisa ter para ser considerado pronto.

| # | Funcionalidade | Descrição |
|---|---|---|
| 1 | Cadastro de usuário | O usuário cria uma conta com nome, e-mail e senha |
| 2 | Login e logout | O usuário entra e sai da conta com segurança |
| 3 | Painel do usuário | Tela principal mostrando os lembretes cadastrados |
| 4 | Criar lembrete | Formulário para escolher data, horário e mensagem |
| 5 | Editar lembrete | O usuário pode alterar um lembrete já criado |
| 6 | Excluir lembrete | O usuário pode apagar um lembrete |
| 7 | Envio de e-mail automático | O sistema envia o e-mail no dia e horário escolhido |
| 8 | Banco de dados | Estrutura para salvar usuários e lembretes |

---

## Funcionalidades planejadas para versões futuras (após a v1.0)

Estas funcionalidades podem ser adicionadas depois da entrega final.

| # | Funcionalidade | Descrição |
|---|---|---|
| 9 | Categorias de lembrete | O usuário pode categorizar os lembretes (trabalho, saúde, estudo…) |
| 10 | Lembretes recorrentes | O usuário pode criar um lembrete que se repete todo mês, semana etc |
| 11 | Notificação antecipada | Enviar um e-mail de aviso um dia antes do lembrete |
| 12 | Histórico de lembretes enviados | Ver os lembretes que já foram disparados |
| 13 | Recuperação de senha | O usuário pode redefinir a senha pelo e-mail |
| 14 | Múltiplos e-mails | Enviar o lembrete para mais de um e-mail |
