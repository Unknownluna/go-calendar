# Visão geral do projeto

## Nome do projeto

Go Calendar

## O que é o sistema?

O Go Calendar é um sistema de lembretes por e-mail. O usuário se cadastra, entra no painel, escolhe uma data e um horário, escreve uma mensagem e o sistema envia um e-mail automático para ele no dia e horário escolhido.

## Objetivo

Criar uma ferramenta simples e acessível para que qualquer pessoa possa cadastrar lembretes e recebê-los por e-mail, sem precisar instalar nada no celular ou computador.

## Público-alvo

Qualquer pessoa que precise ser lembrada de compromissos, tarefas, eventos ou datas importantes.

Exemplos:
- Estudantes que precisam lembrar de provas e trabalhos
- Pessoas que precisam lembrar de consultas médicas
- Profissionais que precisam lembrar de reuniões e prazos

## Funcionalidades principais

- Cadastro e login de usuário
- Painel do usuário com lista de lembretes
- Criação de novo lembrete com data, horário e mensagem
- Edição e exclusão de lembretes
- Envio automático de e-mail no dia e horário escolhido

## Tecnologias

| Tecnologia | Para que serve |
|---|---|
| PHP | Linguagem principal do sistema |
| MySQL | Banco de dados (salva usuários e lembretes) |
| HTML e CSS | Telas do sistema |
| PHPMailer | Envio de e-mails automáticos |
