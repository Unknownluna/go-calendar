---

## Divisão de tarefas

### Bruno é responsável por:
- Criar e manter o repositório no GitHub
- Criar a estrutura do banco de dados (MySQL)
- Programar o sistema de envio de e-mails (PHPMailer)
- Programar o back-end em PHP (cadastro, login, lembretes)
- Publicar o sistema no servidor de produção

### Pedro é responsável por:
- Criar as telas em HTML e CSS
- Fazer os formulários (cadastro, login, novo lembrete)
- Montar o painel do usuário
- Ajudar nos testes do sistema

---

## Como a equipe vai trabalhar

- As tarefas são organizadas no Kanban (arquivo `kanban.md`)
- O prazo de cada etapa está no Gantt (arquivo `gantt.md`)
- A equipe usa a branch `develop` para desenvolvimento e a `main` para produção
- A comunicação é feita por WhatsApp e presencialmente em aula

---

## Ciclos de trabalho (sprints)

O projeto é dividido em 3 sprints, um por bimestre:

| Sprint | Período | Objetivo |
|---|---|---|
| Sprint 1 | 1º bimestre | Planejamento, documentação e estrutura do projeto |
| Sprint 2 | 2º bimestre | Desenvolvimento do sistema (cadastro, lembretes, e-mail) |
| Sprint 3 | 3º bimestre | Testes, correções e publicação da versão 1.0 |
